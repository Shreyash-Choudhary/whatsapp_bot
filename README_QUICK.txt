INSTALLATION (One-Time):
------------------------
Windows: Double-click INSTALL.bat
Mac/Linux: Run ./install.sh

GET API KEY:
-----------
1. Visit: https://console.groq.com/keys
2. Sign up (FREE, no credit card)
3. Create API key
4. Copy it (starts with gsk_...)

GET GROUP LINK:
--------------
1. Open WhatsApp group
2. Tap group name
3. Tap "Invite via link"
4. Copy link

RUN THE APP:
-----------
Windows: Double-click START_APP.bat
Mac/Linux: Run ./start_app.sh

CONFIGURE:
---------
1. Open browser: http://localhost:5000
2. Paste API key
3. Paste group link
4. Click "Save"
5. Click "Initialize WhatsApp"
6. Scan QR code
7. Click "Send Test"
8. Click "Start Scheduler"
